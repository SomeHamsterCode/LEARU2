# Русский ЕГЭ Практика - Игровой формат

## Описание проекта

Это интерактивная платформа для практики заданий ЕГЭ по русскому языку в игровом формате, вдохновлённая Duolingo. Приложение включает:

- Все 19 типов заданий ЕГЭ по русскому языку (без ориентации на текст)
- Геймификацию с системой уровней, опыта, монет и серий
- Возможность создания и настройки персонажа
- Таблицу лидеров
- Статистику прогресса
- Современный дизайн с красной, белой и черной цветовой схемой

## Функциональность

### Задания ЕГЭ
- Реализованы все 19 типов заданий ЕГЭ по русскому языку
- Задания взяты из открытого банка ФИПИ
- Каждое задание имеет объяснение правильного ответа
- Задания неочевидны и требуют глубокого понимания материала

### Геймификация
- Система уровней и опыта
- Монеты за выполнение заданий
- Серии (ежедневная практика)
- Персонализированный персонаж с возможностью настройки
- Таблица лидеров

### Интерфейс
- Современный дизайн с красной, белой и черной цветовой схемой
- Адаптивный интерфейс для разных устройств
- Интуитивно понятная навигация

## Установка и запуск

1. Убедитесь, что у вас установлены Node.js и npm
2. Установите зависимости:
```bash
npm install
```
3. Запустите приложение:
```bash
npm run dev
```

## Компоненты

### Header
- Отображает информацию о пользователе
- Показывает уровень, опыт, монеты и серию
- Содержит навигацию по приложению

### Dashboard
- Главная страница с приветствием
- Отображает статистику пользователя
- Показывает прогресс и недавние задания

### TaskSelector
- Страница с выбором заданий
- Отображает все 19 типов заданий
- Показывает сложность и количество опыта за задание

### TaskPage
- Страница с конкретным заданием
- Поддерживает все типы заданий
- Проверяет ответы и показывает объяснения

### Profile
- Страница профиля пользователя
- Позволяет настраивать персонажа
- Отображает статистику и достижения

### Leaderboard
- Таблица лидеров
- Фильтры по времени
- Отображение топ-3 пользователей

## Цветовая схема

- Белый: #ffffff
- Черный: #000000
- Вспомогательные серые оттенки

## Особенности заданий

Каждое задание разработано на основе открытого банка ФИПИ и проверено на грамотность:
1. Лексика
2. Орфография
3. Пунктуация
4. Морфология
5. Синонимы
6. Антонимы
7. Фразеологизмы
8. Грамматическая основа
9. Слитно, раздельно или через дефис
10. Орфография приставок
11. Правописание суффиксов
12. Лексическая ошибка
13. Синонимичные конструкции
14. Правописание Н и НН
15. Речевые ошибки
16. Пунктуационные ошибки
17. Видо-временные формы глагола
18. Причастие и деепричастие
19. Сложные слова

## Геймификация

- Система уровней: пользователь получает опыт за выполнение заданий
- Монеты: валюта приложения за выполнение заданий
- Серия: награда за ежедневную практику
- Персонаж: пользователь может создать и настроить своего персонажа
- Достижения: специальные награды за выполнение определенных задач
- Таблица лидеров: соревновательный элемент
